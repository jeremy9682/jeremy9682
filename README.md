## Thesis

We’re moving from software that people click through to software that can act.

The next wave of products won’t be defined by chat alone. They’ll be defined by agent systems that can:
- interpret goals
- choose tools and workflows
- execute with guardrails
- learn inside real operational environments

That shift is what I’m building toward across AI agents, markets, automation, and execution infrastructure.

## What I’m building

- agent-native products for trading and market workflows
- execution systems that turn prompts into monitored actions
- automation layers that compress idea → workflow → launch
- full-stack product infrastructure for shipping agent-capable software
