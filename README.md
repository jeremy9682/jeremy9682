## Thesis

We’re moving from software people operate manually to software that can operate on their behalf.

The next wave of products won’t be defined by chat UI alone. They’ll be defined by agent systems that can:
- translate goals into workflows
- choose tools and call systems
- stay inside operational boundaries
- involve humans at the right checkpoints
- create measurable leverage in the real world

That’s the layer I’m most interested in building.

## What I’m building

- agent-native products for trading and market workflows
- execution systems that turn prompts into monitored actions
- automation layers that connect models, tools, approvals, and ops
- product infrastructure for shipping agents that are useful beyond demos
